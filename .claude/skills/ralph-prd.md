# Skill: Ralph PRD - 产品需求文档生成器

## 概述
Ralph 是一个专业的 PRD 文档助手，帮助团队：
1. 收集和整理需求
2. 生成结构化的 PRD 文档
3. 追踪开发进度
4. 验证功能实现

## Ralph PRD 模板

```markdown

## 1. 概述

### 1.1 背景
[描述问题背景和业务需求]

### 1.2 目标
- [主要目标 1]
- [主要目标 2]

### 1.3 范围
[功能范围定义]

## 2. 用户故事

| ID | 用户类型 | 用户故事 | 验收标准 |
|----|----------|----------|----------|
| US-001 | [角色] | [作为...我想...以便于...] | [可验证的标准] |

## 3. 功能需求

### 3.2 详细需求

#### FR-001: [功能名称]
**描述**: [功能详细描述]

**前置条件**: [前置条件]

**后置条件**: [后置条件]

**交互流程**:
1. [步骤1]

## 4. 非功能需求

- 响应时间: < [X]ms

## 5. 技术方案

[数据表设计]
```

## 使用方法

1. **创建 PRD**: `ralph.create_prd(feature_name, requirements)`
2. **更新状态**: `ralph.update_status(fr_id, status)`
3. **运行测试**: `ralph.test(feature_id)`
4. **验证完成**: `ralph.verify(feature_id)`
