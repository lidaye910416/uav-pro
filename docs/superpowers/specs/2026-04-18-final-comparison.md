## 五、异常事件类型

```
┌────────────────────────────────────────────────────────────────────────────────────┐
│                           异常事件类型 (6类)                                            │
├────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  1️⃣ collision   │ 交通事故/碰撞                                                     │
│     特征:         多车聚集、变形车辆、散落物、突然停止                               │
│                                                                                     │
│  2️⃣ pothole    │ 路面塌陷/坑洞                                                    │
│     特征:         路面暗色区域、边缘不规则、积水                                     │
│     检测方式:      帧差法检测变化 → Gemma视觉判断塌陷                               │
│                                                                                     │
│  3️⃣ obstacle   │ 道路障碍物                                                        │
│     特征:         落石、遗撒物、施工围挡、散落货物                                 │
│                                                                                     │
│  4️⃣ parking   │ 异常停车                                                          │
│     特征:         应急车道停车、故障车                                               │
│     检测方式:      帧差法检测静止 + 位置判断                                         │
│                                                                                     │
│  5️⃣ pedestrian │ 行人/动物闯入                                                    │
│     特征:         行人、动物出现在行车道                                            │
│                                                                                     │
│  6️⃣ congestion │ 交通拥堵                                                          │
│     特征:         车辆密度高、移动缓慢                                             │
│                                                                                     │
└────────────────────────────────────────────────────────────────────────────────────┘
```

## 八、为什么选帧差法而非光流法

```
┌────────────────────────────────────────────────────────────────────────────────────┐
│                              选帧差法的原因                                           │
├────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  你的需求分析:                                                                      │
│  ❌ 不需要 TTC                                                                      │
│  ❌ 不需要精确速度/方向                                                            │
│  ❌ 不需要轨迹跟踪                                                                  │
│  ✅ 只需要判断"有没有异常"                                                          │
│                                                                                     │
│  ┌─────────────────────────────────────────────────────────────────────────────┐  │
│  │  帧差法优势:                                                                │  │
│  │  ✅ 计算快 (<10ms/帧)                                                       │  │
│  │  ✅ 实现简单                                                                │  │
│  │  ✅ CPU即可运行，无GPU依赖                                                  │  │
│  │  ✅ 配合Gemma视觉兜底，可检测所有异常(包括静态塌陷)                          │  │
│  └─────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                     │
│  ┌─────────────────────────────────────────────────────────────────────────────┐  │
│  │  光流法劣势 (对你的需求):                                                    │  │
│  │  ❌ 计算量大 (需要GPU才能实时)                                              │  │
│  │  ❌ 实现复杂，需要调参                                                       │  │
│  │  ❌ 优势用不上 (不需要精确速度/方向/TTC)                                     │  │
│  └─────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                     │
│  结论: 帧差法 + Gemma视觉兜底 = 高效、准确、简单                                   │
│                                                                                     │
└────────────────────────────────────────────────────────────────────────────────────┘
```
