## 五、异常事件类型

```
┌────────────────────────────────────────────────────────────────────────────────────┐
│                           异常事件类型 (6类)                                             │
├────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  1️⃣ collision   │ 交通事故/碰撞                                                     │
│     特征:         多车聚集、变形车辆、散落物、突然停止                               │
│                                                                                     │
│  2️⃣ pothole    │ 路面塌陷/坑洞                                                    │
│     特征:         路面暗色区域、边缘不规则、积水                                     │
│                                                                                     │
│  3️⃣ obstacle   │ 道路障碍物                                                        │
│     特征:         落石、遗撒物、施工围挡、散落货物                                 │
│                                                                                     │
│  4️⃣ parking   │ 异常停车                                                          │
│     特征:         应急车道停车、故障车                                               │
│                                                                                     │
│  5️⃣ pedestrian │ 行人/动物闯入                                                    │
│     特征:         行人、动物出现在行车道                                            │
│                                                                                     │
│  6️⃣ congestion │ 交通拥堵                                                          │
│     特征:         车辆密度高、移动缓慢                                             │
│                                                                                     │
└────────────────────────────────────────────────────────────────────────────────────┘
```

## 六、Prompt 体系

### 阶段1: 异常事件识别

```
┌────────────────────────────────────────────────────────────────────────────────────┐
│                         阶段1: 异常事件识别 Prompt                                    │
├────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  System:                                                                            │
│  "你是一个专业的航拍道路图像分析专家。根据图像中的运动分析结果，                    │
│   判断是否存在道路异常事件。"                                                        │
│                                                                                     │
│  User:                                                                              │
│  "图像区域分析结果:                                                                 │
│   - 运动检测: {motion_detection_result}                                              │
│   - 变化区域: {changed_regions}                                                    │
│   - 静止异常: {stationary_objects}                                                  │
│                                                                                     │
│   请判断是否存在以下异常事件:                                                        │
│   1. collision   - 交通事故/碰撞                                                    │
│   2. pothole    - 路面塌陷/坑洞                                                    │
│   3. obstacle   - 道路障碍物                                                      │
│   4. parking   - 异常停车                                                         │
│   5. pedestrian - 行人闯入                                                        │
│   6. congestion - 交通拥堵                                                         │
│                                                                                     │
│   输出JSON: {has_incident, incident_type, confidence}"                                │
│                                                                                     │
└────────────────────────────────────────────────────────────────────────────────────┘
```

### 阶段2: RAG决策输出

```
┌────────────────────────────────────────────────────────────────────────────────────┐
│                         阶段2: RAG决策输出 Prompt                                       │
├────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  System:                                                                            │
│  "你是一个道路安全应急处置专家。根据异常检测结果和相关处置规范，                      │
│   输出标准化的预警信息和处置建议。"                                                  │
│                                                                                     │
│  User:                                                                              │
│  "异常检测结果:                                                                       │
│   - 类型: {incident_type}                                                           │
│   - 置信度: {confidence}                                                            │
│                                                                                     │
│   相关处置规范 (来自RAG检索):                                                        │
│   {rag_context}                                                                      │
│                                                                                     │
│   请输出:                                                                            │
│   - risk_level: 低/中/高/紧急                                                       │
│   - title: 预警标题(10字内)                                                          │
│   - description: 详细描述(30字内)                                                    │
│   - recommendation: 处置建议(40字内) ← 核心输出                                     │
│   - confidence: 置信度"                                                             │
│                                                                                     │
└────────────────────────────────────────────────────────────────────────────────────┘
```
